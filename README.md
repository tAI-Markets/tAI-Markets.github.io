# tAI Markets Website

The official landing page for tAI Markets — the world's first clearing house for AI inference.

## Overview

This repository contains the GitHub Pages site for tAI Markets, a decentralized CDP/CCP clearing house that transforms AI computation into tradeable, collateralized financial assets.

## Features

- **Modern Design**: Dark theme with animated background and live transaction ticker
- **Responsive**: Works on desktop, tablet, and mobile devices
- **Fast**: Static HTML/CSS with no JavaScript dependencies
- **SEO Optimized**: Meta tags and semantic HTML structure

## Local Development

To run this site locally:

```bash
# Clone the repository
git clone https://github.com/tAI-Markets/tAI-Markets.github.io.git
cd tAI-Markets.github.io

# Serve with any static file server
python3 -m http.server 8000

# Or use Jekyll
bundle install
bundle exec jekyll serve
```

Then open http://localhost:8000 in your browser.

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.
